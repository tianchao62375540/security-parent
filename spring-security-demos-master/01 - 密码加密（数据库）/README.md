## 使用手册

1、创建数据库
```
create database `security01` default character set utf8mb4 collate utf8mb4_general_ci;
```

2、执行 `user.sql`

3、执行 `mvn spring-boot:run`

4、注册账号 http://localhost:8080/register

5、登录 http://localhost:8080/login
